Instruções para execução:

Para processar as operações, o programa precisa receber a lista de instruções desejadas.
Assim, adicione a entrada desejada no arquivo "in.txt", execute o comando ./run.sh e o resultado
do processamento será salvo num arquivo chamado "result.txt".

Atenção, os seguintes requisitos são necessários para rodar o programa:
Ubuntu 18.04 ou superior
Python 3.6 ou superior
Um terminal qualquer com suporte a bash.
